# Components
<br/><br/>

``코로나 예방접종 센터 조회 서비스`` 프로젝트의 공통 컴포넌트 Storybook 입니다.

* ``Comp_Button.vue``: 버튼
* ``Comp_Card.vue``: 카드
* ``Comp_GaugeBar.vue``: 막대 게이지
* ``Comp_IconButton.vue``: 아이콘 버튼
* ``Comp_SelectBox.vue``: 콤포박스
* ``Comp_TransitionOpacity.vue``: ``fade-out/fade-in`` 효과 컴포넌트



<br/><hr/>


# Components 개요
<br/><br/>


## 01. ``Comp_Button.vue``

``border-radius``가 약간 둥근 스타일의 버튼 컴포넌트 입니다.

``click`` 기능을 가지며, ``mouseenter`` 및 ``mouseleave`` 효과와 더불어 ``enter`` 와 ``leave`` 이벤트가 발생 됩니다.Button

[Comp_Button](?path=/story/components-comp-button--basic)



<br/><hr/><br/>



## 02. ``Comp_Card.vue``

상단 이미지와 하단 컨텐츠로 구성된 카드 컴포넌트 입니다.

``click`` 이벤트를 사용할 수 있습니다.

[Comp_Card](?path=/story/components-comp-card--basic)



<br/><hr/><br/>



## 03. ``Comp_GaugeBar.vue``

비율 표시를 위한 게이지 컴포넌트 입니다.

``click`` 시, 게이지가 다시 그려지는 animation이 동작 됩니다.

[Comp_GaugeBar](?path=/story/components-card-gaugebar--basic)



<br/><hr/><br/>



## 04. ``Comp_IconButton.vue``

문자열이 없고 ``아이콘`` 으로 구성된 원형 버튼 입니다.

``click`` 시, ``@click`` 이벤트가 발생 됩니다.

(Storybook의 FontAwesome 문제로 인해, Prototype의 아이콘이 출력되지 않고 있습니다. - 프로젝트 상에는 정상 동작 합니다.)

[Comp_IconButton](?path=/docs/components-comp-iconbutton--basic)



<br/><hr/><br/>



## 05. ``Comp_SelectBox.vue``

``selectBox`` 컴포넌트 입니다.

선택지와 값을 다르게 사용할 수 있도록 ``I_Comp_SelectBox`` interface 타입을 사용 합니다.

선택된 값이 없을 때와 있을 때의 우측 아이콘이 다르게 표시 됩니다.

선택된 값이 있을 때, 우측 아이콘을 클릭하면 선택된 값을 해제할 수 있습니다.

(Storybook의 FontAwesome 문제로 인해, Prototype의 아이콘이 출력되지 않고 있습니다. - 프로젝트 상에는 정상 동작 합니다.)

[Comp_SelectBox](?path=/story/components-comp-selectbox--basic)
