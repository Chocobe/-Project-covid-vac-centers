# Components
<br/><br/>

``코로나 예방접종 센터 조회 서비스`` 프로젝트의 공통 컴포넌트 Storybook 입니다.

* ``Comp_Button.vue``: 버튼
* ``Comp_Card.vue``: 카드
* ``Comp_GaugeBar.vue``: 막대 게이지
* ``Comp_IconButton.vue``: 아이콘 버튼
* ``Comp_SelectBox.vue``: 콤포박스
* ``Comp_TransitionOpacity.vue``: ``fade-out/fade-in`` 효과 컴포넌트
